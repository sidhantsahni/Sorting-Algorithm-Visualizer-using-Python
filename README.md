# Sorting Algorithm Visualizer

## 📌 Description

This project is a Python-based interactive visualization tool that
demonstrates how different sorting algorithms work in real-time using
animated bar graphs.

## 🚀 Features

-   Real-time visualization of sorting algorithms\
-   Supports:
    -   Bubble Sort\
    -   Selection Sort\
    -   Insertion Sort\
    -   Merge Sort\
    -   Quick Sort\
-   Random array generation\
-   Smooth animations using Matplotlib

## 🛠 Technologies Used

-   Python\
-   Matplotlib\
-   Random module

## ▶️ How to Run

``` bash
python main.py
```

## 🎯 Purpose

This project helps students and beginners understand sorting algorithms
visually and intuitively.

## 🔮 Future Improvements

-   Add more sorting algorithms\
-   GUI using Streamlit or Tkinter\
